# AI Resume Parser

An AI-powered resume analysis platform built using FastAPI and React.

## Features

- Resume Upload (PDF/DOCX)
- AI Match Score
- Skill Matching
- Missing Skill Detection
- Recruiter Summary Generation
- Modern React Frontend

## Tech Stack

Frontend:
- React
- Axios
- CSS

Backend:
- FastAPI
- Sentence Transformers
- Python

## Run Backend

cd backend
uvicorn main:app --reload

## Run Frontend

cd frontend
npm install
npm run dev
