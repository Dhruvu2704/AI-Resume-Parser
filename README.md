# AI Resume Screening & Ranking System

An AI-powered resume analysis platform that helps recruiters evaluate candidates using semantic matching and skill-gap analysis.

## Features

- Resume PDF/DOCX Parsing
- Semantic Resume Matching
- Skill Gap Analysis
- Recruiter Insights
- FastAPI Backend
- Sentence Transformers

## Tech Stack

- Python
- FastAPI
- Sentence Transformers
- Scikit-Learn
- PyMuPDF
- python-docx

## API Endpoints

- /upload-resume
- /job-description
- /match-score
- /skill-analysis
- /recruiter-analysis
